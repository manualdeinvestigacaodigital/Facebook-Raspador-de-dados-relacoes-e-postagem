# FB B35A1 44.21.28 — UI de exportação e vídeo no HTML

Esta é uma **cópia candidata derivada da base estável 44.21.27**, que permanece congelada e não foi modificada.

Escopo exclusivo:

1. reorganização visual do seletor de formatos para manter visíveis os quatro botões de ação;
2. recuperação estritamente vinculada da URL de vídeo do Reel quando a requisição GraphQL do vídeo contém o ID da rota ativa;
3. captura de vídeo visível dentro da superfície exata de POST/VIDEO;
4. inclusão, no HTML, do player e do botão **Baixar ou abrir vídeo completo** quando uma URL reutilizável tiver sido efetivamente observada.

Não houve alteração no coletor de relações 44.21.27, em comentários, respostas, reações, PHOTO, scroll, navegação, service worker ou exportadores de dados não relacionados a vídeo.
